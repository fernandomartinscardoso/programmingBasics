# 📚 Foundational C++ Projects and Study 💻

[![Static Badge](https://img.shields.io/badge/C%2B%2B-6771B0?logo=c%2B%2B&logoColor=white&labelColor=585858)](https://isocpp.org/)
[![GitHub License](https://img.shields.io/github/license/fernandomartinscardoso/cppBasics?color=5B9C73)](https://github.com/fernandomartinscardoso/cppBasics/blob/main/LICENSE)

This repository, __cppBasics__, is a collection of fundamental projects, code examples, and study materials designed for learning, testing, and mastering the C++ programming language. It is a resource for anyone starting their journey with C++ or looking for concise, runnable examples of core language features.

The intent is to provide a clean, organized environment to explore basic to intermediate C++ concepts without the complexity of large frameworks or advanced libraries.

## 🎯 Goals

The primary goals of this repository are to:

- __Provide Reference Code__: Offer clear, simple, and runnable C++ code (`.cpp` files) that illustrates specific language features, syntax, and standard library usage.
- __Support Study__: Serve as a personal knowledge base and study log, documenting essential C++ concepts, best practices, and common pitfalls.
- __Enable Rapid Prototyping__: Allow users to quickly test and verify understanding of new C++ topics using the existing structure.
- __Share Knowledge__: Create a resource that others can easily clone, explore, and use as a learning companion.

## 📂 Repository Structure

The project is structured to separate source code examples from detailed theoretical articles and notes.

```
cppBasics/
├── articles/           # Detailed written explanations, notes, and theoretical articles (e.g., Markdown or PDF files) on C++ topics.
├── basicCodes/         # Source code files (.cpp) containing simple, runnable examples for core C++ concepts.
├── LICENSE             # The text of the MIT License.
└── README.md           # This file.
```
